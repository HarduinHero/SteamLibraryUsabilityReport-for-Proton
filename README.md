SLURP aim is to generate a usability report of your steam library with proton using ratings from ProtonDB.

**This file exists in other languages** : [FranÃ§ais](/doc/README.fr.md)

## Constraint

* JUST A SCRIPT - (no need to make a web app for that)
* Do not require a public steam account.

## How to use
Full how to with images [here](/doc/full_how_to.en.md).

### Use .exe 
1. Download the last .exe release of this project. 
1. On your browser connect to your steam account on https://store.steampowered.com.
1. Access your game library, ``YourUsername > Games``.
1. Select the ``All Games (x)`` tab.
1. Save the web page with ``Ctrl+S`` as ``target.html`` next to ``SLURP_xx.exe``.
1. Execute ``SLURP_xx.exe``.
1. Your report has appeared !!

### Use as python script
1. Download / Clone this repo.
1. On your browser connect to your steam account on https://store.steampowered.com.
1. Access your game library, ``YourUsername > Games``.
1. Select the ``All Games (x)`` tab.
1. Save the web page with ``Ctrl+S`` as ``target.html`` next to ``SLURP.py``.
1. Execute ``SLURP.py``.
1. Your report has appeared !!
